# checkyourlist

A minimalist library of Canadian paperwork checklists — designed for clarity, speed, and peace of mind.

## What this project includes
- Tax documents checklist  
- Citizenship application checklist  
- Passport application checklist  
- Permanent residency (PR) checklist  
- Self‑employed tax checklist  
- NEXUS application checklist  

All checklists are available as clean, premium PDFs.

## Coming soon

- Camping essentials  
- Flying and airport prep

- Moving day basics  
- First‑apartment setup  
- Seasonal home maintenance  

These will follow the same minimalist, stress‑free format.

## Purpose
